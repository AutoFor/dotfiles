# Google 日本語入力 キーマップ設定

親指シフトユーザー向けの Google 日本語入力キーマップ。

親指シフトでは**無変換キーを右親指シフトキー**として使用しているため、入力中・変換中の無変換キーは何もしない設定にしている。

## 方針

**Neovim で IME の on/off が分かりにくい問題**を解決するため、Mac 風のキー割り当てに変更している。

Mac JIS キーボードでは「英数キー → IME off」「かなキー → IME on」が独立しているため直感的に切り替えられる。
Windows でも同様の操作感を再現するため、無変換・変換キーを専用の on/off キーとして使用する。

## キー割り当て

| キー | 状態 | 動作 |
|---|---|---|
| 無変換 | 未入力時 (Precomposition) | IME Off |
| 無変換 | 入力中・変換中 | 何もしない |
| 変換 | DirectInput | IME On |
| 変換 | その他 | ひらがなモードに切り替え |
| Ctrl+U | 入力中・変換候補中 | 日本語 ↔ アルファベット トグル |

## インポート方法

1. Google 日本語入力の設定を開く
2. 「キー設定」→「キー設定の選択」→「編集」
3. 「インポート」から `google-keymap.txt` を選択
