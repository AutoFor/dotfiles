# 自分専用ショートカット＆コマンドメモ（WezTerm / Zsh）

WSL2 + WezTerm + Zsh 環境で使っている、自分がよく忘れるショートカットとコマンドをまとめたメモです。

## 環境

- **ターミナル**: WezTerm（Windows）
- **シェル**: Zsh on WSL2 (Ubuntu)
- **プラグイン**: zoxide, fzf, ghq

各ツールの設定ファイルは https://github.com/AutoFor/dotfiles を参照してください。

---

## WezTerm

Leader キーは `Ctrl+q`（2秒タイムアウト）。

### Workspace

| ショートカット | 動作 |
|--------------|------|
| `Ctrl+q → w` | workspace 一覧を表示して切り替え |
| `Ctrl+q → $` | 現在の workspace 名を変更 |
| `Ctrl+q → Shift+W` | 新規 workspace を作成 |

### Tab

| ショートカット | 動作 |
|--------------|------|
| `Cmd+t` | 新規タブを開く |
| `Cmd+w` | 現在のタブを閉じる |
| `Ctrl+Tab` | 次のタブへ移動 |
| `Ctrl+Shift+Tab` | 前のタブへ移動 |
| `Cmd+1`〜`Cmd+9` | タブ番号で切り替え（9 は最後のタブ） |
| `Ctrl+q → {` | タブを左に移動 |
| `Ctrl+q → }` | タブを右に移動 |

### Pane

| ショートカット | 動作 |
|--------------|------|
| `Ctrl+q → r` | ペインを左右に分割 |
| `Ctrl+q → d` | ペインを上下に分割 |
| `Ctrl+q → x` | 現在のペインを閉じる |
| `Ctrl+q → h/l/k/j` | 左/右/上/下のペインへ移動 |
| `Ctrl+Shift+[` | ペインを番号で選択 |
| `Ctrl+q → z` | 現在のペインをズーム（トグル） |
| `Ctrl+q → s` → `h/l/k/j` | ペインサイズを調整（Enter で終了） |
| `Ctrl+q → a` → `h/l/k/j` | ペイン移動モード（1秒タイムアウト） |

### コピーモード（vi ライク）

| ショートカット | 動作 |
|--------------|------|
| `Ctrl+q → [` | コピーモードを起動 |
| `h/j/k/l` | 左/下/上/右に移動 |
| `w / b / e` | 単語ごとに前/後/末尾に移動 |
| `^ / $` | 行の最初の文字 / 行末に移動 |
| `0` | 行の左端に移動 |
| `gg / G` | バッファ先頭 / 末尾に移動 |
| `Ctrl+f / Ctrl+b` | 1ページ下/上にスクロール |
| `Ctrl+d / Ctrl+u` | 半ページ下/上にスクロール |
| `v` | 文字単位の選択モード |
| `V` | 行単位の選択モード |
| `Ctrl+v` | ブロック選択モード |
| `y` | 選択範囲をクリップボードにコピー |
| `Enter` | 選択範囲をコピーしてモード終了 |
| `q / Escape / Ctrl+c` | コピーモードを終了 |

### その他

| ショートカット | 動作 |
|--------------|------|
| `Cmd+c` | クリップボードにコピー |
| `Cmd+v` | クリップボードから貼り付け |
| `Ctrl++` | フォントサイズを拡大 |
| `Ctrl+-` | フォントサイズを縮小 |
| `Ctrl+0` | フォントサイズをリセット |
| `Cmd+p` / `Ctrl+Shift+p` | コマンドパレットを開く |
| `Ctrl+Shift+r` | 設定を再読み込み |
| `Alt+Enter` | フルスクリーン切り替え |

## Zsh キーバインド

### ghq

| ショートカット / コマンド | 動作 |
|------------------------|------|
| `Ctrl+G` | 管理リポジトリを fzf で絞り込んで即 `cd` |
| `ghq create <name>` | 新規リポジトリを作成（`~/ghq/` 以下に配置） |
| `ghq get <URL>` | リポジトリをクローン（`~/ghq/` 以下に配置） |
| `ghq list` | 管理リポジトリを一覧表示 |
| `ghq list -p` | フルパスで一覧表示 |
| `ghq root` | ghq のルートディレクトリを表示 |

---

## Zsh コマンド

### zoxide（スマートな `cd`）

| コマンド | 動作 |
|---------|------|
| `z <キーワード>` | 履歴から部分一致して `cd` |
| `zi` | fzf で履歴からディレクトリをインタラクティブ選択 |
| `zoxide query <キーワード>` | ジャンプせずに候補リストだけ確認 |

よく行くディレクトリは `z` に任せると `cd` を打つより圧倒的に速い。

| コマンド | 動作 |
|---------|------|
| `cfd` | カレントディレクトリ直下のフォルダを fzf で選んで `cd` |
| `wcd "C:\..."` | Windows パスを WSL パスに変換して `cd` + Claude Code 起動 |

## Neovim

Leader キーは `Space`。

### 起動・終了

| コマンド | 動作 |
|---------|------|
| `nvim <ファイル>` | ファイルを指定して起動 |
| `nvim .` | カレントディレクトリを開いて起動 |
| `:qa` | 全ウィンドウを閉じて終了 |
| `:qa!` | 未保存の変更を破棄して強制終了 |
| `:wqa` | 全ファイルを保存して終了 |

### ファイルツリー（nvim-tree）

| キー | 動作 |
|-----|------|
| `Ctrl+n` | ファイルツリーを開閉 |
| `<leader>er` | カーソル下のディレクトリをルートに変更して再表示 |
| `gy` | カーソル下のファイルの絶対パス（Linux）をクリップボードにコピー |
| `gW` | カーソル下のファイルの Windows パスをクリップボードにコピー（WSL2用） |
| `gr` | カーソル下のファイルの相対パスをクリップボードにコピー |
| `a` | 新規ファイル/ディレクトリを作成（末尾に `/` でディレクトリ） |
| `d` | カーソル下のファイル/ディレクトリを削除（フォルダにカーソルを合わせて実行） |
| `Ctrl+]` | カーソル下のフォルダを nvim-tree 内部のルートに変更（`:pwd` は変わらない） |

### ファイル内検索

| キー / コマンド | 動作 |
|---------------|------|
| `/キーワード` | カーソル以降を前方検索 |
| `?キーワード` | カーソル以前を後方検索 |
| `n` | 次のマッチへ移動 |
| `N` | 前のマッチへ移動 |
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |
| `:noh` | 検索ハイライトを消す |

### 行番号移動

| キー / コマンド | 動作 |
|---------------|------|
| `:行番号` | 指定行へジャンプ（例: `:42`） |
| `行番号G` | 指定行へジャンプ（例: `42G`） |
| `gg` | ファイルの先頭へ |
| `G` | ファイルの末尾へ |
| `数字j` | 現在行から N 行下へ移動（例: `10j`） |
| `数字k` | 現在行から N 行上へ移動（例: `10k`） |

### Claude Code / Codex 連携

| キー | 動作 |
|-----|------|
| `<leader>ac` | Claude Code を開閉（`:Claude` でも可） |
| `<leader>ab` | 現在のファイルを Claude Code に追加 |
| `<leader>as` | 選択範囲を Claude Code に送信（ビジュアルモード） |
| `<leader>cx` | Codex を開閉（`:Codex` でも可） |
| `<leader>av` | 縦分割（vertical）で Claude Code を開く（MCP連携あり） |
| `<leader>ah` | 横分割（horizontal）で Claude Code を開く |

### スクロール

| キー | 動作 |
|-----|------|
| `Ctrl+f` | 1画面分下にスクロール |
| `Ctrl+b` | 1画面分上にスクロール |
| `Ctrl+d` | 半画面分下にスクロール |
| `Ctrl+u` | 半画面分上にスクロール |
| `gg` | ファイルの先頭に移動 |
| `G` | ファイルの末尾に移動 |

### ウィンドウ移動

| キー | 動作 |
|-----|------|
| `<leader>h` | 左のウィンドウに移動 |
| `<leader>l` | 右のウィンドウに移動 |
| `<leader>j` | 下のウィンドウに移動 |
| `<leader>k` | 上のウィンドウに移動 |
| `<leader>w` | 次のウィンドウに移動（順番に切り替え） |
| `<leader>c` | 現在のウィンドウを閉じる |
| `:q` | 現在のウィンドウを閉じる |

### Markdown プレビュー（glow）

| キー | 動作 |
|-----|------|
| `<leader>md` | 右ペインで glow プレビューを開く（markdown ファイルのみ） |
| `Alt+q` → `<leader>c` | glow プレビューを閉じる（ターミナルモード抜け → ウィンドウ閉じる） |

### アウトライン（outline.nvim）

| キー | 動作 |
|-----|------|
| `<leader>o` | アウトラインを開閉（Markdown の見出し一覧／LSP シンボル） |

### Claude Code 再起動

Neovim 内の Claude Code は「閉じて再度開く」ことで再起動できる（`claudecode.nvim` の既知仕様）。

| キー / コマンド | 動作 |
|---------------|------|
| `<leader>ac`（Claude Code フォーカス中） | Claude Code パネルを閉じる（＝セッション終了） |
| `<leader>ac`（閉じた後） | 新しい Claude Code プロセスを起動（＝再起動） |
| `:ClaudeCodeClose` | パネルを明示的に閉じる |

ターミナル内の Claude Code プロセス自体を再起動したい場合：

| コマンド | 動作 |
|---------|------|
| `exit` または `Ctrl+D` | Claude Code セッションを終了 |
| `claude` | 新規セッションで Claude Code を起動 |
| `claude -r <セッション名>` | 指定セッションで再開 |

### ターミナルモード (Calode Code , Codex)

| キー | 動作 |
|-----|------|
| `Alt+q` | ターミナルモード → ノーマルモードに戻る |

### ファイル操作

| キー / コマンド | 動作 |
|---------------|------|
| `:e` | ファイルをディスクから再読み込み（リロード） |
| `:w` | 保存 |
| `:q` | 閉じる |
| `:wq` | 保存して閉じる |

### 元に戻す／やり直す

| キー | 動作 |
|-----|------|
| `u` | 元に戻す（Undo） |
| `Ctrl+r` | やり直す（Redo） |

### 文字削除

| キー | 動作 |
|-----|------|
| `x` | カーソル下の1文字を削除 |
| `X` | カーソルの左の1文字を削除 |
| `dw` | 単語を削除（次の単語の先頭まで） |
| `db` | 単語を後方に削除 |
| `dd` | 行全体を削除 |
| `D` | カーソルから行末まで削除 |
| `d0` | カーソルから行頭まで削除 |
| `diw` | 単語全体を削除（空白は残す） |
| `daw` | 単語全体＋前後の空白を削除 |
| `di"` | `"..."` の中身を削除 |
| `da"` | `"..."` ごと削除（引用符含む） |
| `c` 系 | `d` と同じ範囲を削除してインサートモードへ |
